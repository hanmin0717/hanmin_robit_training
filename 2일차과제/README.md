1. 직관적이고 시각적인 차트 자체를 직접 화면에 그려보고 싶다는 생각이 들었습니다. 그래서 라이브러리인 matplotlib를 추가로 공부하여, 개발자들이 가장 선호하는 프로그래밍 언어 데이터를 막대그래프로 그리는 프로그램을 새로 만들어 보게 되었습니다.

2. 사용한 외부 라이브러리 및 버전
openpyxl 3.1.5 버전과 matplotlib 3.8.0 버전을 설치하여 사용했습니다.

3. 라이브러리에 대한 이해와 배운 점
첫 번째로 사용한 openpyxl은 파이썬 코드로 엑셀을 제어하는 도구입니다.

두 번째로 사용한 matplotlib는 파이썬에서 데이터를 보기 좋은 그림이나 차트로 변환해 주는 시각화 도구입니다. 데이터를 기반으로 깔끔한 막대그래프를 생성하는 것을 알았습니다. 또한 그래프 위에 정확한 수치 텍스트를 출력하여 보는 사람의 이해도을 높이는 법도 알게되었습니다.
