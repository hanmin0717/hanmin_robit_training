1. 프로젝트 개요와 라이브러리를 바꾼 이유
처음에는 파이썬으로 데이터를 다루고 시각화해보고 싶어서, 엑셀을 자동으로 다루는 openpyxl 라이브러리로 주가 정리 프로그램을 짜보았습니다.

하지만 엑셀 파일 생성에 그치지 않고 조금 더 직관적이고 시각적인 차트 자체를 직접 화면에 그려보고 싶다는 생각이 들었습니다. 그래서 다른 라이브러리인 matplotlib를 추가로 공부하여, 개발자들이 가장 선호하는 프로그래밍 언어 데이터를 막대그래프로 그리는 프로그램을 새로 만들어 보게 되었습니다.

2. 사용한 외부 라이브러리 및 버전
openpyxl 3.1.5 버전과 matplotlib 3.8.0 버전을 설치하여 사용했습니다.

3. 라이브러리에 대한 이해와 배운 점
첫 번째로 사용한 openpyxl은 파이썬 코드로 엑셀을 제어하는 도구입니다. 행과 열 데이터를 지정해서 거래대금을 연산하고, 엑셀 수식인 =AVERAGE()를 심어 평균을 계산하며, 자체 차트 기능을 시트에 삽입하는 방법을 익혔습니다.

두 번째로 사용한 matplotlib는 파이썬에서 데이터를 보기 좋은 그림이나 차트로 변환해 주는 시각화 도구입니다. 데이터를 기반으로 깔끔한 막대그래프를 생성하고, 각 막대마다 구분이 잘 가도록 파스텔톤 컬러를 직접 매핑하는 방법을 배웠습니다. 또한 그래프 위에 정확한 수치 텍스트를 출력하여 보는 사람의 시인성을 높이는 법도 익힐 수 있었습니다.

4. 프로그램 기능 및 실행 방법
샤페론 주가 데이터 엑셀 자동화 프로그램
샤페론의 실제 주가 데이터를 바탕으로 shaperon_stock.xlsx 파일을 자동으로 만듭니다. 반복문으로 각 행의 거래대금을 자동 계산해 채워 넣고, 맨 하단에 =AVERAGE() 함수를 입력해 평균 종가를 구합니다. 그리고 openpyxl.chart 모듈로 종가 추이 그래프를 엑셀 시트 안에 자동으로 그려 넣습니다.
